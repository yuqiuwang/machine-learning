### 2018.7.22

添加了 **pic\_cnv\_extract.py**
脚本对图片进行预处理
脚本能从 800X1280 的染色体图片切割目标CNV位置，并压缩至指定大小（默认300X300）
使用压缩后的图片进行深度学习

用法：
    **`python pic_cnv_extract.py -i ./sample1_chr2.png -o ./sample1_2_36173502_38514591.png -p chr2,36173502,38514591`**
